### 檔案清單（C:\投資報告\）

| 檔案 | 功能 |
|---|---|
| `full_stock_analysis.py` | 完整個股分析引擎 v3（877 行，主力） |
| `daily_report.py` / `daily_report_v2.py` | 每日損益報告（整合版） |
| `stock_analysis.py` | 個股分析引擎 v2（舊版，留作備援） |
| `market_summary.py` | 市場動態摘要 |
| `dashboard.py` | HTML 儀表板產生器（含房產/基金區塊） |
| `insurance.py` | 保險管理模組 |
| `notify.py` | Telegram 推播模組 |
| `query.py` | 歷史查詢選單（含選項 8 房產、9 基金） |
| `setup_db.py` | 資料庫建立（執行一次） |
| `setup_real_estate_fund.py` | 房產/基金 DB 建立 |
| `fix_owner.py` | 資產歸屬修正 |
| `auto_context.py` | 雲端自動更新 CONTEXT.md（VM 上執行） |
| `cloud_analysis.py` | 雲端版個股分析（VM 上執行） |
| `investment.db` | SQLite 主資料庫 |
| `analysis/` | 每日 JSON 分析存檔 |
| `reports/` | 每日 HTML 報告 |

### SQLite 資料庫（`investment.db`）資料表

- `holdings` — 現有持倉
- `realized_pnl` — 已實現損益
- `pledge_records` — 質借紀錄
- `insurance` — 保險記錄
- `daily_snapshots` — 每日損益快照
- `transactions` — 交易記錄
- `fx_rates` — 匯率記錄
- `stock_analysis_history` — 每日個股分析歷史
- `real_estate` — 房產（含 `owner` 欄位區分個人/旭源）
- `funds` — 基金（含 `owner` 欄位區分個人/旭源）
